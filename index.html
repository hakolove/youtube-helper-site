YouTube Content Management Helper

Description:

This desktop tool is designed to help YouTube content managers improve workflow efficiency. It allows users to manage video metadata (titles, descriptions, subtitles, thumbnails) and Content ID assets directly from the interface, without needing to manually operate through YouTube Studio or CMS.

The tool integrates with internal project management systems (such as Feishu) and is built purely for productivity use within a small content operation team. It does not share, analyze, or sell any user data.

  • Batch update of video metadata (title, description, thumbnail)
  • Subtitles translation and upload
  • Asset and policy management for YouTube Content ID partners
  • Local secure storage of authentication tokens (no backend server)

This application requests only the https://www.googleapis.com/auth/youtubepartner scope, which is strictly used to enable copyright asset and policy control functionalities.
